Modern C++: move && forward

2023/12/22 Modern_C++

std::move()

该函数将传入的参数变成将亡值（会清空该参数），同时返回该函数的右值，方便移动构造，实现所有权交接

其本质并不是移动，而是强制类型转换，把参数强制转换成为右值

ps. 函数返回局部对象时请不要 std::move ，你的编译器会自动做优化 (RVO) ，如果使用可能会破坏优化

std::forward()

problem

在使用万能引用时，由于不清楚最终参数会被匹配为什么引用，而 C++ 中，所有引用都是左值，当外部传入一个右值时，在模板内部的参数是右值引用，当在模板内部传入该引用，其会被认为是左值，从而匹配左值引用，所有需要 forward 进行转发

function

std::forward()实现了完美转发（即正确调用函数），左值引用传左值引用的重载函数，右值引用传右值引用的重载函数

其本质是有选择的类型转换，把右值引用转化为右值，左值引用则保持不变

其原理其实是 T&& 中的 T 保存了传入时的参数状态：左值或者右值

difference

std::forward() 有选择的右值转换

std::move() 强制右值转换
