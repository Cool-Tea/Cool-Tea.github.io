F

Description

https://codeforces.com/contest/1926/problem/F

Solution

由于本题为 7×77 \times 77×7 的固定大小，可以使用枚举的方法完成

继续分析可以发现，在最坏情况下最多需要去掉8个黑色格子就可以使整个grid没有X，也就是说，本题的最大搜索深度为8，此时估计枚举量为 C498C^8_{49}C498​ 约为4e8，这显然是非常极限的

由于X的特殊性，其只受对角线上的方格影响，不难发现，从(1,1)开始，将按照对角线方式移动所能到达的格子抽离出来，在这些格子上枚举不影响未抽离的格子的X存在性

这时，分别枚举处理两边，单边的枚举量为 C254C^4_{25}C254​ 大约为1e4，两边加起来也不过2e4，运算量大大降低。

G

Description

https://codeforces.com/contest/1926/problem/G

Solution

如果把P点与S点看成水泵，P点与S点的水不同，那么题目所要求的就是让水向根节点流，期间不能发生水混合的情况

定义状态：假设 dp[i][j]dp[i][j]dp[i][j] 表示在i节点处及其子节点处不会发生液体混合时，液体情况为j时，所需的最少筑墙数，j可取[0,1,2]，分别表示没有液体流过、没有S液体流过、没有P液体流过的情况。

注意，当i节点为水泵节点时，认为有液体流过且只流其泵出的液体

转移方程：

  • C节点：$$dp[i][0]=\sum \min (dp[child][0], dp[child][1]+1, dp[child][2]+1)$$$$dp[i][1]=\sum \min (dp[child][0], dp[child][1], dp[child][2]+1)$$$$dp[i][2]=\sum \min (dp[child][0], dp[child][1]+1, dp[child][2])$$
  • P节点：$$dp[i][0]=dp[i][2]=\infin$$$$dp[i][1]=\sum \min (dp[child][0], dp[child][1], dp[child][2]+1)$$
  • S节点：$$dp[i][0]=dp[i][1]=\infin$$$$dp[i][2]=\sum \min (dp[child][0], dp[child][1]+1, dp[child][2])$$

注意，没有S/P液体流过表示要么流其他液体，要么不流液体，所以在转移时，必须模糊考虑这两种情况，由于三种情况可以互相弥补未考虑的情况，所以其包含了全部情况
